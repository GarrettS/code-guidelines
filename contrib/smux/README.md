# smux Dev Workflow Spike

Experimental contributor tooling for reducing manual agent relay during Web XP development.

## Status

Not started. This directory is a placeholder for the smux/tmux-bridge integration spike.

## Goal

Replace the file-based handoff protocol with direct agent-to-agent messaging via tmux panes. Reduce "check your inbox" relay from the human.

## Non-goal

- General orchestration engine
- User-facing feature
- Replacing single-agent workflows
